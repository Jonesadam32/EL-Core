# Paste workspace

Use this folder to paste PHP code from the staging site (e.g. EL Core 1.22.13).

## How to use

- **Single file:** Paste the full plugin PHP into `el-core-main.php` (overwrite the placeholder).
- **Multiple files:** Create files or folders here as needed (e.g. `includes/`, `modules/`, etc.) and paste into them.

After pasting, we can compare with the repo, merge into `el-core/`, or create an `old-versions/v1.22.13/` backup.
